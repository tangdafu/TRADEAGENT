🚀 快速开始
═══════════════════════════════════════════════════════════════════════════

  1. 安装依赖
     $ pip install -r requirements.txt

  2. 配置API（编辑 .env 文件）
     LLM_API_BASE_URL=https://api.asxs.top
     LLM_API_KEY=sk-your-api-key

  3. 运行分析
     $ python src/main.py
